### Troubleshooting

If issues occur:

1. Check logs:

   ```powershell
   docker compose logs [service-name]
   ```

2. Restart specific service:

   ```powershell
   docker compose restart [service-name]
   ```

3. Full restart:

   ```powershell
   docker compose down
   docker compose up -d
   ```

4. View errors only:
   ```powershell
   docker compose logs | Select-String "ERROR|FATAL|Exception"
   ```
