**Title:** `Commands` (Jitsi Meet SDK executeCommand)

**Purpose:** Catalog of commands that can be executed via `api.executeCommand(command, ...arguments)`. These commands control meeting behavior, participant actions, and UI state. Use this reference to understand command names, input parameters, and their effects.

**Notes:**
- All commands are executed via `api.executeCommand(command, ...arguments)`.
- Some commands require moderator privileges.
- Parameter descriptions include type information and optional flags.
- Commands are case-sensitive; use exact names as listed.
